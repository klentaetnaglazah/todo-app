# Todo List App

Веб-приложение для управления списком задач. Позволяет создавать, редактировать и удалять задачи с возможностью фильтрации и поиска.

## Стек

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Fastify
- **Database**: SQLite (sql.js)

## Запуск локально

```bash
git clone https://github.com/klentaetnaglazah/todo-app
cd todo-app
npm install
npm start
```
Приложение будет доступно по адресу http://localhost:3000
